# FixBuddy Frontend - Static Version

This is a static HTML version of the FixBuddy Neighbourhood Service Marketplace frontend, ready for deployment to Vercel, Netlify, or any static hosting service.

## 🚀 Quick Deploy to Vercel

### Option 1: Using Vercel CLI (Recommended)
```bash
# Install Vercel CLI
npm install -g vercel

# Login to Vercel
vercel login

# Deploy
vercel --prod
```

### Option 2: Using Vercel Website
1. Go to [vercel.com](https://vercel.com)
2. Click "New Project"
3. Import your GitHub repository or upload the files
4. Deploy automatically

### Option 3: Using Netlify
```bash
# Install Netlify CLI
npm install -g netlify-cli

# Login to Netlify
netlify login

# Deploy
netlify deploy --prod --dir=.
```
